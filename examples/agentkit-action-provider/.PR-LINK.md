# AgentKit action provider — PR snapshot

This directory is a snapshot of the proposed `x402station` action provider for [`@coinbase/agentkit`](https://github.com/coinbase/agentkit) — submitted upstream as [PR #1154](https://github.com/coinbase/agentkit/pull/1154).

While the PR is in review, you can drop these files into your local AgentKit fork at `typescript/agentkit/src/action-providers/x402station/` and add `export * from "./x402station";` to `typescript/agentkit/src/action-providers/index.ts`. After merge, just `import { x402stationActionProvider }` from `@coinbase/agentkit` directly.
